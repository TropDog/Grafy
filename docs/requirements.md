# Project Requirements

This project relies on several Python packages to function correctly. Below is a list of the required dependencies along with a brief description of their purpose.

## Dependencies

| Package        | Version  | Description                                                         |
|----------------|----------|---------------------------------------------------------------------|
| `networkx`     | 3.1      | A library for creating and manipulating complex networks and graphs. |
| `pytest`       | 7.4.2    | A testing framework to ensure your code works correctly.            |
| `pytest-cov`   | 4.1.0    | A plugin for measuring code coverage during tests.                  |
| `matplotlib`   | Latest   | A plotting library for creating visualizations.                     |
| `mkdocs-material` | Latest | A theme for MkDocs that provides a modern and responsive design.    |
| `mkdocstrings[python]` | Latest | A MkDocs plugin for generating API documentation from docstrings. |

## Installation

To install the required dependencies, run the following command:

```bash
pip install -r requirements.txt
